# This program requires Python 3.12 (or higher)

Download Python from the official website:
[https://www.python.org/](https://www.python.org/)

## List of imported libraries (file `requirements.txt`)

```txt
tkcalendar
pandas
matplotlib
openpyxl
markdown
```

Some libraries are already standard and included in the basic Python installation:

- `tkinter` is the standard library for creating graphical interfaces.
- The libraries `webbrowser`, `json`, `csv`, `re`, `subprocess`, `datetime`, `hashlib`, `uuid`, and `pathlib` are standard and do not require separate installation.

## Installing Dependencies

If you use pip to install dependencies, simply add them to the requirements.txt file and install them with the command:

```bash
pip install -r requirements.txt
```

## Launch the application

Run main_ru.py for the Russian version of the program.
