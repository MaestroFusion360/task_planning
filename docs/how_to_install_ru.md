# Для работы программы необходима установка Python 3.12 (или выше)

Скачайте Python с официального сайта:
[https://www.python.org/](https://www.python.org/)

## Список импортируемых библиотек (файл `requirements.txt`)

```txt
tkcalendar
pandas
matplotlib
openpyxl
markdown
```

Некоторые библиотеки уже являются стандартными и входят в базовую установку Python:

- `tkinter` — стандартная библиотека для создания графических интерфейсов.
- Библиотеки `webbrowser`, `json`, `csv`, `re`, `subprocess`, `datetime`, `hashlib`, `uuid`, `pathlib` — являются стандартными и не требуют отдельной установки.

## Установка зависимостей

Если вы используете `pip` для установки зависимостей, просто добавьте их в файл `requirements.txt` и установите с помощью команды:

```bash
pip install -r requirements.txt
```

## Запуск приложения

Запустите main_ru.py для русскоязычной версии программы
